# Retail-Sales-PowerBI-Dashboard.
## Project overview:
This project analyzes retail sales data using Microsoft Excel and Power BI to understand sales performance, customer behavior, and product trends. The dashboard provides interactive visualizations that help identify key business insights.
## Tools Used:
Microsoft Excel – Data Cleaning and Preparation
Power BI – Data Visualization and Dashboard Creation
GitHub – Project Documentation and Version Control Dataset
## The dataset: 
Order ID
Order Date
Region
Category
Product Name
Sales
Profit
Discount
Customer Segment
## Dashboard Features:
The Power BI dashboard includes the following visualizations:
Total Sales, Profit, and Orders (KPI cards).
Sales by Region
Monthly Sales Trend
Sales Distribution by Category
Sales by Customer Segment
Top Products by Orders
Discount vs Profit Analysis
Regional Sales Map
## Key Insights:
The East region generated the highest sales revenue.
Technology products contributed significantly to overall sales.
The Consumer segment accounted for the largest share of purchases.
Higher discounts tend to reduce overall profit margins.
